    • Primary memory is the one of the most important component of the computer architecture . Here the data and instruction are temporarily store and it’s mainly used for processing.
    • CPU can directly access primary memory.
    • They are volatile memory, which mean they lose the data when computer is turned off.

    • There are 2 types of RAM :
      • S-RAM (Static Random Access Memory) → S-RAM can hold the data as long as the power availability is there. It is refreshed simultaneously to store the present information. It is made with CMOS technology. It contains 4 to 6 transistors and it also uses clocks. It does not require a periodic refresh cycle due to the presence of transistors. Although SRAM is faster, it requires more power and is more expensive in nature.
      • D-RAM (Dynamic Random Access Memory) →
    • Difference between S-RAM and D-RAM :
      S-RAM D-RAM
      1. They are made of transistor.1. They are made of capacitor.
      2. Transistor do not required to be refresh periodically.2. Capacitor do required to be refresh periodically.
      3. S-RAM are faster than D-RAM.3. D-RAM are slower than S-RAM.
      4. More expensive than D-RAM. 4. Cheaper than S-RAM.
      5. They consume more power than D-RAM.5. They consume less power than S-RAM.
      6. They are used to implement cache memory.6. They are used to implement RAM.
  • ROM (Read Only Memory) → This are special type of primary memory where we can not change the data, It is a nonvolatile memory which means when the computer power is turned off the data is not lost.
    • There are 3 Types of ROM :
      • PROM (Programmable Read Only Memory) → This type of ROM can be program by the user, The process of programming ROM is Known as burning. Once the data is program in the ROM it can not be modify.
      • EPROM (Erasable Programmable Read Only Memory) → This type of ROM Overcomes the problem of PROM by allow the user to program as well as erase the data in chip. We erase the data from EPROM using Ultraviolet Rays.
      • EEPROM (Electrically Erasable Programmable Read Only Memory) → This type of ROM overcomes the problem of erasing the ROM using UV rays by using electricity. EEPROM known as flash memory.

5. Computer Language and Software: 

Algorithms →

Planning a program before actually implementing it using a programming language. like C, C++, Java, Python, etc.

An algorithm is a step by step description of how to arrive at a solution of a given problem.

  • Characteristic of an algorithm →
    • Clear and Unambiguous: The algorithm should be unambiguous. Each of its steps should be clear in all aspects and must lead to only one meaning.
    • Well-Defined Inputs: If an algorithm says to take inputs, it should be well-defined inputs. It may or may not take input.
    • Well-Defined Outputs: The algorithm must clearly define what output will be yielded and it should be well-defined as well. It should produce at least 1 output.
    • Finite-ness: The algorithm must be finite, i.e. it should terminate after a finite time.
    • Feasible: The algorithm must be simple, generic, and practical, such that it can be executed with the available resources. It must not contain some future technology or anything.
    • Language Independent: The Algorithm designed must be language-independent, i.e. it must be just plain instructions that can be implemented in any language, and yet the output will be the same, as expected.
    • Input: An algorithm has zero or more inputs. Each that contains a fundamental operator must accept zero or more inputs.
    • Output: An algorithm produces at least one output. Every instruction that contains a fundamental operator must accept zero or more inputs.
    • Definiteness: All instructions in an algorithm must be unambiguous, precise, and easy to interpret. By referring to any of the instructions in an algorithm one can clearly understand what is to be done. Every fundamental operator in instruction must be defined without any ambiguity.
    • Finiteness: An algorithm must terminate after a finite number of steps in all test cases. Every instruction which contains a fundamental operator must be terminated within a finite amount of time. Infinite loops or recursive functions without base conditions do not possess finiteness.
    • Effectiveness: An algorithm must be developed by using very basic, simple, and feasible operations so that one can trace it out by using just paper and pencil.

    • There are 3 types of data transmission modes :
      • simplex → Here two interconnected devices exchange information or transmitted data in one direction only that is allows sender to receiver.
      • Half-duplex → Here two interconnected devices wish to exchange data alternatively that is sender to receiver and vise-versa, but transmitted data from one direction at a time. It is also known as semi duplex mode.
      • Duplex or Full-duplex → Here two interconnected devices wish to exchange data and information in both direction simultaneously that is data transmitted is both direction at a certain time. Here data flow is by-directional in nature.

  • Data Transmission Media→
    • Guided Media →

      It is also known as wires signals being transmitted are directed in a narrow pathway by using physical link.

      • The feature of guided media are →
        1. High speed
        1. Secure
        1. They are mainly use for sort distance communication
      • There are 3 major types of guided media →
        1. Twisted Pair
        1. So-Axial Cable
        1. Optical Fiber
    • Twisted Pair →

    → A twisted pair consist of two insulated copper wire then twisted around each other to reduce interference from nearby wires.

    A twisted pair is used in sort distance communication (1km). They are cheap, easy to install but they are easily pickup noises which causes error during transmission.

    • So-Axial Cable →

      → A co-axial cable consist of a central copper wire which is surrounded by a PVC insulator, The PVC insulator is enclosed by a copper mesh which reduces interference from outside environment.

      They provide higher bandwidth than twisted pair. They allow long distance communication (10-100km). Further more they have higher immunity from noise.

    • Optical Fiber →

      → A fiber optic cable is a cable that contains one or more strands of thin, hair-like glass or plastic fibers that can transmit light. These fibers are used to carry data over long distances at high speeds. It is similar to an electrical cable, but uses light pulses instead of electrical currents to transmit information.

    • Unguided Media →

      Unguided media is wireless communication that uses signals such as radio waves, microwaves, infrared and satellite communication. It provides flexible and mobile communication but can suffer from interference and limited bandwidth.

      • The feature of guided media are →
        1. Flexibility
        1. Mobility
        1. Allowing devices to communicate without the need for physical connections.
        1. Susceptible to interference.
      • Types of Unguided Media →
        • Radio waves: Used for radio and television broadcasting, wireless networking, and mobile phone communication.
        • Microwaves: Employed in satellite communication, Wi-Fi, and radar systems.
        • Infrared: Used for short-range communication like TV remote controls and data transfer between nearby devices.
        • Satellite signals: Enable GPS navigation, satellite TV, and long-distance data transmission.
  • Topologies →
    • Network Topologies →
      • The physical arrangement of a network of the layout of the network is called the network topology.
    • Types of network topologies →
      • Bus topology
      • Mesh topology
      • Ring topology
      • Star topology
      • Tree topology
      • Hybrid topology

  • Bus topology →
    • A bus topology is a multipoint connection of rows because every node is connected to a single cable.

      The main cable is the backbone of the network. In bus topology since the data can travel only trough one path so there is a chance of data collision.

      Advantage Disadvantage
      1. Simple architecture.1. A crack in a signal can be break the network.
      2. Installation cost is very low.2. Only a single signal can be transmitted at a time.
      3. It is very easy to add a new node.
  • Mesh topology →
    • In a mesh topology every device has a single dedicated line to every other device in the network.

      In a mesh topology if there are n numbers of node then there will be “n(n-1)/2” number of cables/units there for every device must have n-1 ports.

      • NOTE → No. of cable = n(n-1)

        Where n = no. of nodes.

      Advantage Disadvantage
      1. There is no traffic problem as there is a dedicated point to point links for every computer.1. Installation cost is high.
      2. If a link fails, it does not effect the entire network2. large amount of cable has required.
      3. Maintenance is difficult.
      4. It is very difficult to convert a new node.
  • Star topology →
    • In star topology every node communicate with another node in the network through a central hub.

      The central hub acts as a point to point link in between two nodes. The hub also acts as a traffic control of data transmission, which enables a secure transmission of data.

      Advantage Disadvantage
      1. The star topology provides a more secure war of data transmission.1. Star topology requires a central hub which is more expensive than a normal node.
      2. It is easy to setup and maintain.2. If the hub goes down the entire network goes down.
      3. It requires less cable as compare to mesh topology.
      4. Unlike bus topology there is no data collision.
  • Ring topology →
    • A ring topology is a network arrangement were devices are connected in a circular path. We can imagine ring topology as a bus topology were the end points are connected to form a ring.

      In a ring topology data can flow in clockwise direction or anticlockwise direction.

      In this type of topology we assume that the data has been transmitted successfully when as acknowledgement massage is received by the sender.

      Advantage Disadvantage
      1. If a ring topology is unidirectional there is no chance of data collision.1. Due to the unidirectional nature of ring topology a data packet must trough all other nodes.
      2. The cost of installation is less.2. It is very difficult to ass a new node to a network.
      3. There is no need of costly system like hub to manage the network.3. It is very difficult to find a fault in the transmission medium.
      4. Even is there is a single crack in a cable the whole network become in-operational.
  • Tree topology →
    • A Tree topology is a combination of star and bus topology. It provides a hierarchical structure where the top most node is called the root. Each node is gives rise to two or more nodes, they are called the children nodes those nodes that does not have any child node are called the leaf node.
  • Hybrid topology →
    • A hybrid topology is a combination of any two topology. In general we combine two topologies that can compliment each other, which means the drawback of one topology can be rectified with another topology.
